# ユーザー登録機能と投稿機能

## バージョン
* Python: 3.11
* Django: 4.2
* Django REST framework: 3.15.2
* Node.js: 18.20.4(v20ではfirebaseが動かなかったため)
* firebase: 13.23.1

## 環境設定
* react-django-appディレクトリで以下を実行
`npm install firebase-tools`

### フロントエンド側
* frontendディレクトリで以下を実行
`npm install`

* firebaseのデプロイ
`firebase deploy`

## 実行
`docker compose up --build`
